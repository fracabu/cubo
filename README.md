<a name="english"></a>
## :gb: English

### Overview

A lightweight and interactive web application for viewing basic 3D scenes, optimized for low-end hardware. Features interactive controls, performance monitoring, and energy-saving optimizations.

### Features

- **3D Scene Viewer** - Basic geometry visualization
- **Interactive Controls** - Rotate, zoom, pan
- **Performance Monitoring** - FPS counter
- **Energy Saving** - Optimized for low-end devices
- **Lightweight** - Minimal dependencies

### Quick Start

```bash
git clone https://github.com/fracabu/cubo.git
cd cubo
# Open index.html in browser
```
